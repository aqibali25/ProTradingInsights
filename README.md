# Pro Trading Insights

A single-screen, 100vh Next.js landing page inspired by the provided reference layout, rebuilt with a new navy, teal, and gold visual system and updated content.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Main files

- `app/page.jsx` — page structure and content
- `app/globals.css` — full responsive design
- `public/pro-trading-insights-logo.jpeg` — supplied logo
